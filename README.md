# DDN-DE experimental-data

Supplementary experimental data and comprehensive statistical results for the paper "DDN-DE: A Dynamic Diverse Neighborhoods-based Differential Evolution Algorithm with Self-learning Control Parameters".

Due to the manuscript's length limits, this repository provides the following extended experimental results to ensure the reproducibility and transparency of our research:

---

## 📊 Experimental Data Overview

The supplementary data are organized into two main parts:

### 1. Parameter Sensitivity Analysis
* **Format:** `Parameter Sensitivity Analysis.md`
* **Description:** This section provides the exhaustive statistical results for the control parameters, including the neighborhood size parameter ($CP$), superior learning probability ($spb$), and orthogonal learning probability ($opb$).
* **Note:** These extended data tables correspond to **Section 4.2 (Parameter Sensitivity Analysis)** of the main manuscript.

### 2. Comparative Accuracy Analysis
* **Format:** `Algorithm Accuracy Analysis.docx`
* **Description:** This part contains the comprehensive Mean and Standard Deviation (Std) values for the proposed DDN-DE algorithm compared against eight DE variants (JADE, SHADE, cnEpSin, CUSDE, TDE, RNGDE, DE-EXP, and MIDE).
* **Scope:** The performance is assessed across 30 benchmark functions from the CEC2017 suite at dimensions **D=30**, **D=50**, and **D=100**.
* **Note:** These results correspond to **Section 4.3 (Comparison with other Algorithms)** of the main manuscript.
